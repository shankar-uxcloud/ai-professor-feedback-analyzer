About Professor Feedback Analyzer

This prototype uses AI analysis to help college students interpret professor feedback, identify strengths and weaknesses, and receive concrete improvement tips.

Key features include:

  • Upload environment for PDF/DOCX/TXT content and direct comment input
  • AI report with structured strengths, weaknesses, key learning areas, and grade prediction
  • Visual dashboard with charts and grade tracking
  • History storage and PDF export for documentation
  • Dark mode, responsive layout, and professional design

Technology stack: Node.js backend, OpenAI integration (configurable), Chart.js, jsPDF, and modern vanilla JS frontend.
